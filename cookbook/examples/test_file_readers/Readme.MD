# Document Viewer

This is a Streamlit-based web application that allows users to upload documents or specify a file path to view the contents of the following document types: PDFs, CSVs, text files, and Word documents.

Steps to run: 
1. pip install -r cookbook/examples/test_file_readers/requirements.txt
2. pip install -e . (from the phidata folder)
3. streamlit run cookbook/examples/test_file_readers/app.py

Let me know if you run into problems or need anything else to test!
